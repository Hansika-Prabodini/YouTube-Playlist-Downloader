# YouTube Playlist Downloader

Welcome to the YouTube Playlist Downloader project! This repository provides two powerful and easy-to-use tools for downloading videos from YouTube: a **Graphical User Interface (GUI)** application and a **Command-Line Interface (CLI)** script.

## What Makes This Project Special

Both tools are designed to make downloading YouTube content simple and efficient. Whether you prefer a visual interface or the power of the command line, we've got you covered. The tools leverage the robust **yt-dlp** video downloader to ensure reliable and high-quality downloads.

## Key Features

### GUI Application
- **User-Friendly Interface**: Easy to use with an intuitive graphical window
- **Playlist Support**: Load entire YouTube playlists and see all video titles
- **Flexible Downloads**: Download specific videos or all videos from a playlist
- **Visual Progress**: Real-time progress bars for each downloading video
- **Custom Save Locations**: Choose where to save your downloaded videos
- **Convenient Controls**: Right-click paste support and cancel options

### CLI Script
- **Terminal-Based**: Work directly from your command line interface
- **Playlist Loading**: List all videos from any YouTube playlist URL
- **Smart Selection**: Choose specific videos, ranges, or download everything
- **Real-Time Progress**: View yt-dlp's download progress directly in your terminal

## Powered by yt-dlp

Both applications use **yt-dlp**, a powerful and actively maintained video downloader that ensures compatibility with YouTube and provides reliable downloads with support for various video and audio formats.
